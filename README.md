# CheckList Pro 📋

App de inspeção de equipamentos com geração de relatório em PDF.

## Funcionalidades

- ✅ Adicionar/remover equipamentos
- ✅ Status: **Conforme**, **N/A** ou **Não Conforme**
- ✅ Campo de observações para N/A e Não Conforme
- ✅ Barra de progresso em tempo real
- ✅ Dados salvos automaticamente (localStorage)
- ✅ Geração de PDF profissional com jsPDF
- ✅ PWA — funciona offline e pode ser instalado no celular
- ✅ Design responsivo (mobile-first)

## Como usar localmente

Abra o `index.html` diretamente no navegador. Nenhum servidor necessário.

## Como hospedar no GitHub Pages

1. Crie um repositório no GitHub (pode ser público ou privado)
2. Faça upload dos arquivos `index.html` e `manifest.json`
3. Vá em **Settings → Pages**
4. Em **Source**, selecione `main` branch e pasta `/root`
5. Clique em **Save**
6. Aguarde 1-2 minutos e acesse `https://SEU-USUARIO.github.io/NOME-DO-REPO`

## Tecnologias

- HTML5 + CSS3 + JavaScript vanilla
- [jsPDF](https://github.com/parallax/jsPDF) para geração de PDF
- Google Fonts (DM Sans + Syne)
- localStorage para persistência de dados
- Web App Manifest (PWA)
